# OpenBrain

A minimalist, distraction-free "Second Brain" to instantly capture and semantically retrieve fleeting digital artifacts.

## Overview
OpenBrain is designed for researchers, designers, and curators suffering from information overload. It employs a "Strict Swiss Execution" design strategy—rigorous grids, high-contrast monochrome hierarchy, structural whitespace, and glassmorphic modal overlays.

## Stack
- Next.js 16 (React 19)
- Tailwind CSS v4
- Supabase / Supermemory

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design System Note
Check `app/globals.css` for Tailwind v4 theme CSS token overrides (Typography, Palette, Borders).
